# Vue3地图前端架构

## 项目结构
```
frontend/
├── src/
│   ├── assets/          # 静态资源
│   ├── components/      # Vue组件
│   ├── App.vue         # 根组件
│   └── main.js         # 入口文件
├── index.html          # HTML模板
├── vite.config.js      # Vite配置
└── package.json        # 依赖配置
```

## 技术栈
- Vue 3 Composition API
- Vite 构建工具
- Axios HTTP客户端
- 现代CSS Grid布局

## 开发命令
```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 生产构建
npm run build

# 预览构建结果
npm run preview
```

## 功能特性
- 响应式现代化UI设计
- 实时车辆位置显示
- 自动数据刷新
- 移动端适配
- 组件化架构
